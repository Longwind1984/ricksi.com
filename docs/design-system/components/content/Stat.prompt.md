Workbench stat block — small mono letterspaced label on top, big white tabular number below, with an optional small unit and a faint mono footnote. Numbers are always real or flagged with the sample badge.

```jsx
<Stat label="今日 TOKEN 用量" value="1,247,832" size="big" sub="全口径（含缓存读写 41%）· 其中模型输出 89K" />
<Stat label="本周" value="8.7M" unit="tokens" />
```
