# Microsoft

> 大模型时代的"管道与渠道"巨头。1975 年 Bill Gates 与 Paul Allen 创立，PC 操作系统时代缔造者。在生成式 AI 浪潮中通过 2019 年起对 [OpenAI](/kb/ai-公司与产品/openai/) 累计 130 亿美元以上的战略投资，绑定了 GPT 模型族的算力 (Azure) 与渠道 (Copilot 全产品线)，成为"基础设施 + 分发"双轨上 AI 时代收益最稳定的传统巨头。但与 [OpenAI](/kb/ai-公司与产品/openai/)、[Anthropic](/kb/ai-公司与产品/anthropic/) 等纯研究实验室的产品打磨能力相比，Microsoft 的 AI 产品形态（Copilot in Office / Windows / GitHub）被广泛批评为"组织功能性的胜利、产品体验的代差"。

## 公司画像
- **创始基因**：操作系统与办公套件起家，靠"标准化分发 + 企业渠道"建立现代软件业最稳的护城河。Satya Nadella 2014 年接任后从"Windows everywhere"转向"cloud first / AI first"。
- **资本结构**：纳斯达克市值 3 万亿美元量级（2025–2026），与 Apple / Nvidia 并列全球三大科技市值巨头。AI 相关业务以 Azure OpenAI、GitHub Copilot、Microsoft 365 Copilot 三轨变现。
- **AI 战略**：以"独家算力供应商 + 全产品分发渠道"绑定 OpenAI，把 GPT 模型族包装为 Copilot 嵌入 Office / Windows / Edge / Bing / GitHub / Dynamics / Power Platform 等全套产品线。同时对外保留"我们仍有自研 AI"叙事——MAI 团队 (Mustafa Suleyman 主导)、Phi 系列开源小模型、Inflection AI 团队收编。
- **政策角色**：在 AI 监管议题上立场偏"积极合规 + 推动行业治理框架"，与 [OpenAI](/kb/ai-公司与产品/openai/) 的 Altman 频繁同台。Brad Smith (President) 是行业政策对话的主要发言人。

## 关键产品与时间线
- 2019.7 首次对 [OpenAI](/kb/ai-公司与产品/openai/) 投资 10 亿美元，Azure 成为 OpenAI 独家算力供应商。
- 2021.6 GitHub Copilot 公测，基于 OpenAI Codex；2022.6 GA。这是大模型时代第一个百万级 DAU 的生产力 AI 产品。
- 2023.1 追加投资约 100 亿美元；2023.2 Bing Chat (后改名 Copilot) 集成 GPT-4。
- 2023.3 Microsoft 365 Copilot 发布，把 GPT-4 接入 Word/Excel/PowerPoint/Outlook/Teams 全套办公产品。
- 2023.11 Ignite 大会：Copilot 品牌统一覆盖 Windows / Edge / Office / GitHub / Dynamics。
- 2024.3 收编 Inflection AI 团队，CEO Mustafa Suleyman 出任 Microsoft AI (MAI) 负责人。
- 2024 Phi-3 系列开源小模型发布，对外标榜"小模型路线"。
- 2025 Copilot+ PC 品牌联合 OEM 厂商推动端侧 NPU 大规模铺货。
- 2025.11 Ignite 后转向 "Agent Factory" 叙事，把 Copilot Studio 重定位为企业 Agent 工厂。
- 2026 当下 Copilot 全线渗透但用户满意度持续低于 [ChatGPT](/kb/ai-公司与产品/chatgpt/) / [Claude](/kb/ai-公司与产品/claude/)——Rick 在 20260304-AI产品体验的代差与组织问题 与 20260304-微软Copilot和谷歌Gemini的产品体验问题 中详细记录。

## 与 OpenAI 的合作-竞争关系
- **合作面**：Azure 是 OpenAI 训练 / 推理的独家算力供应商；OpenAI 获得资本与上层政治资源，Microsoft 获得最强模型的优先接入权。
- **竞争面**：Copilot in Office 与 [ChatGPT](/kb/ai-公司与产品/chatgpt/) 在企业市场直接互搏；OpenAI 上线 Apps Platform / Custom GPTs / 直接面向 B 端销售订阅，逐步绕开 Microsoft 渠道。
- **2023.11 政变事件**：OpenAI 董事会短暂罢免 Sam Altman 时，Microsoft 公开宣布"无论 Altman 在哪我们都跟随"——这是合作关系从"伙伴"显性化为"控股股东级影响力"的转折点。
- **2025+ 渐行渐远**：OpenAI 启动自有数据中心（与 Oracle / SoftBank 合作的 Stargate 项目），开始解除 Azure 独家绑定；Microsoft 也对外加大 Phi 自研投入与 Anthropic API 接入。

## 同行对比
- **vs. Google / Alphabet**：Google 自有计算 + 搜索分发 + 端云 + 自研 [Gemini](/kb/ai-公司与产品/gemini/)，垂直整合度更高；Microsoft 走"投资 + 渠道嵌入"模式，灵活但被外部研究节奏掣肘。
- **vs. Amazon**：Amazon 走"算力 + 投资 [Anthropic](/kb/ai-公司与产品/anthropic/) + 自研 Nova 模型"路线，比 Microsoft 更分散下注，但产品落地更弱。
- **vs. Meta**：Meta 走"自研 Llama + 全开源"路线，与 Microsoft 的"投资闭源模型 + 全产品 Copilot 化"形成战略两极。
- **vs. Apple**：Apple 走"端侧 + 隐私 + 联合 OpenAI/Anthropic 外援"路线，Microsoft 全力 cloud + AI 一体化。两家在 PC + 移动端 AI 入口将持续短兵相接。

## 对 Rick 的价值
- **AI 产品组织失败的典型样本**：Copilot 全产品铺面，但用户满意度、留存、口碑都不如 ChatGPT / Claude / Cursor 等纯 AI 产品。Rick 在多篇笔记中把 Microsoft Copilot 作为"组织结构与产品体验之间的代差"案例——大公司即便拿到最强模型 + 最广渠道，依然做不出最好用的 AI 产品。这一观察对 通往 AI PM 之路 的目标公司选择有直接影响。
- **理解 OpenAI 真实治理结构的窗口**：要看清 [OpenAI](/kb/ai-公司与产品/openai/) 的实际权力分布，必须看 Microsoft 的资本与算力杠杆。Sam Altman / Satya Nadella 的双人轴心是 AI 行业最重要的 power couple。
- **企业 AI 落地的 incumbent 视角**：Rick 在 AI 产品形态 与 [Agent 产品化](/kb/ai-工程化与落地架构/m206-agent-产品化-记忆机制与技术进展/) 中分析的"AI 嵌入既有 SaaS vs 重新发明"两种路线，Microsoft 是前者最大规模的实证样本。

## 关键人物
- **Satya Nadella**（CEO）：印度裔，2014 接任后完成 Microsoft 从"Windows-first"到"cloud-first / AI-first"的战略转型；与 Altman 形成 AI 时代最重要的 CEO 双人轴。
- **Bill Gates**（创始人 / 前 CEO）：仍以慈善基金会身份对 AI 议题持续发声，2023 后明确表态 GPT-4 是 1980 年图形 UI 之后他见过的最重要技术突破。
- **Brad Smith**（President / Vice Chair）：政策与法务负责人，AI 治理对外发言人。
- **Kevin Scott**（CTO）：Microsoft–OpenAI 合作主推手，负责 Azure AI / Copilot 技术栈整合。
- **Mustafa Suleyman**（Microsoft AI CEO）：[DeepMind](/kb/ai-公司与产品/gemini/) 联创、Inflection AI CEO，2024 被 Microsoft 收编主导 MAI 自研团队与 Copilot 消费端产品。
- **Scott Guthrie**（EVP, Cloud + AI Group）：Azure 业务负责人，AI 基础设施工程主推手。
